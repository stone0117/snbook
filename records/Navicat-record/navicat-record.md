| CMD-2                              | View                                   |
| CMD-3                              | Function/SQLite Index                  |
| CMD-4                              | MySQL/MariaDB Event/SQLite Trigger     |
| CMD-5                              | Query                                  |
| CMD-6                              | Backup/Oracle Data Pump                |
| CMD-7                              | Schedule                               |
| CMD-8                              | Model                                  |
| **CMD-L**                          | Query Log                              |

| SHIFT-CMD-T                        | Data Transfer                          |

| SHIFT-CMD-R                        | Restore Backup                         |
| CTRL-CMD-G                         | Setup Schedule                         |
| CTRL-CMD-O                         | Erase Schedule                         |
