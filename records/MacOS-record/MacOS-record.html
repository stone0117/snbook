mds_stores直接占用92%内存，导致应用崩溃！！！！电脑业很热啊变得

已解决 如下：

mds、mds_stores、mdworker这几个进程都是为了Spotlight索引而疯狂的，解决方案：
关闭
sudo mdutil -a -i off
打开
sudo mdutil -a -i on
